# Строитель дорог

Игра-головоломка, в которой нужно соединить города оптимальной сетью дорог. Игрок должен найти минимальное остовное дерево для заданного набора городов.

## Особенности

- Три уровня сложности с разным количеством городов
- Визуальное отображение весов дорог
- Подсветка возможных соединений при выборе города
- Проверка оптимальности решения
- Интуитивно понятный интерфейс

## Требования

- Python 3.8 или выше
- Pygame 2.5.2

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd <repository-name>
```

2. Создайте виртуальное окружение (опционально, но рекомендуется):
```bash
python -m venv venv
```

3. Активируйте виртуальное окружение:
- Windows:
```bash
venv\Scripts\activate
```
- Linux/Mac:
```bash
source venv/bin/activate
```

4. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

```bash
python -m src.main
```

## Управление

- ЛКМ (Левая кнопка мыши) - выбрать город или построить дорогу
- ПКМ (Правая кнопка мыши) - вернуться в меню выбора уровней

## Правила игры

1. Выберите уровень сложности
2. Нажмите на город, чтобы выбрать его
3. Нажмите на другой город, чтобы построить дорогу между ними
4. Соедините все города дорогами
5. Попробуйте найти решение с минимальной общей длиной дорог

## Структура проекта

```
src/
├── main.py              # Точка входа в игру
├── constants.py         # Константы и настройки
├── game/               # Игровая логика
│   ├── city.py         # Класс города
│   ├── road.py         # Класс дороги
│   ├── level.py        # Класс уровня
│   └── levels_data.py  # Данные уровней
└── ui/                 # Пользовательский интерфейс
    ├── button.py       # Класс кнопки
    └── level_select.py # Экран выбора уровня
```

## Алгоритм

Игра использует алгоритм Крускала для нахождения минимального остовного дерева. Это гарантирует, что оптимальное решение всегда существует и может быть найдено.
